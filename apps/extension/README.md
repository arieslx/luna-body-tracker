# Luna Body Tracker Extension

一个离线优先的 Chrome 新标签页身体记录板。

当前扩展使用 Vite + React + TypeScript 重构。源码在 `src/`，构建产物在 `dist/`，Chrome 加载未打包扩展时请选择 `dist/` 文件夹。

## 开发

```bash
pnpm --filter @luna-body-tracker/extension dev
```

## 打包

```bash
pnpm --filter @luna-body-tracker/extension build
```

## 加载方式

先执行上面的打包命令，然后：

1. 打开 Chrome。
2. 进入 `chrome://extensions/`。
3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择构建产物文件夹：`apps/extension/dist`。
6. 打开新标签页即可使用。

## 当前功能

- 心情
- 喝水
- 睡眠
- 运动及自定义子类目
- 体重
- 今日食物池
- 拉屎
- 备注
- 模块隐藏/显示
- 模块上下排序
- 自定义食物项，食物池使用吃过/没吃过记录
- `records.jsonl` 导出
- `summary.md` 导出
- 导出摘要包含最近记录概览
- 周视图

当前新标签页只显示操作控件，不显示统计数据；历史和结构化数据通过导出查看。

## 当前交互

- 日期显示为 `yyyy/mm/dd 周几`，左右按钮切换前后日期。
- 睡眠支持按住并滑过格子选择。
- 今日食物池内置项带 emoji，自定义食物在分割线下方单独显示。
- 运动分为主类目「有氧 / 无氧」和自定义运动区。
- 拉屎和备注位于页面底部，拉屎使用最小模块。
- 第一排为心情、喝水、睡眠；喝水和睡眠在右侧同宽同高竖向排列。
- 第二排为今日食物池，包含早餐、午餐、晚餐、加餐和食物备注。
- 第三排为运动，使用 emoji + 文字。
- 第四排为拉屎、体重和备注。
- 标题统一为 17px；喝水 8 个水杯同排显示；食物池餐次 placeholder 为「想写就写」。
- emoji + 文字按钮为左右结构，文字 19px。
- 睡眠格子横向滚动；食物池内置「海鲜」替代「其他」。
- 睡眠格子为独立多选，点击可选中/取消，不再自动连续填充。
- 高亮色按模块色系加深：喝水为蓝色，运动为粉色。
- 体重输入显示 `KG` 单位。
- 拉屎和体重右上角有眼睛开关，可将数据区高斯模糊隐藏，再次点击恢复显示；隐藏状态按日期独立保存。
- 顶部「周视图」按钮可切换到笔记本样式周计划，按周一到周日纵向展示每日记录。
- 周视图每一天分为日期/心情、睡眠/餐次或食物类别、运动、拉屎、体重、喝水六列。

## 字体

- 全局优先使用 `SF Mono`
- 中文字符由浏览器按系统字体回退显示

## 数据存储

- 每日记录保存在 IndexedDB：`luna_body_tracker_db`
- 设置保存在 `chrome.storage.local`
- 数据默认只保存在本机，不上传、不同步

## 说明

缺失字段表示没有记录，不代表没有发生。
`0` 只有在用户明确记录为 0 时才表示 0。
