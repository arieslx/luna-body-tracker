# Yun Tracker StickS3 Assets

These are the canonical MVP assets. Do not resize, crop, recolor, regenerate,
or rename them unless explicitly requested.

## Required Files

- `day.png`, `night.png`: home island backgrounds, `135x240`.
- `day_bg.png`, `night_bg.png`: soft text/summary backgrounds, `135x240`.
- `bubble_left.png`, `bubble_right.png`: platform action prompts.
- `icon_mood.png`, `icon_stress.png`, `icon_poop.png`, `icon_food.png`,
  `icon_water.png`, `icon_sleep.png`, `icon_sport.png`, `icon_oracle.png`.
- `platform_mood.png`, `platform_poop.png`, `platform_food.png`,
  `platform_sleep.png`, `platform_sport.png`, `platform_oracle.png`.
- `lan_stand_0.png`, `lan_stand_1.png`.
- `lan_walk_0.png`, `lan_walk_1.png`.

## Notes

- PNG alpha is allowed and expected for icons, bubbles, and Lan frames.
- Code should adapt to the actual image dimensions rather than assuming every
  icon is exactly `20x20`.
- Device target path is `/flash/res`.
